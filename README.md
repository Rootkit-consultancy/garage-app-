# Garage App (Expo SDK 54)

This repo is an **Expo / React Native** project (SDK 54) implementing a mechanic booking flow:

- Home / Find Mechanic
- Mechanic Profile
- Booking Details & Schedule
- Live Tracking
- In-app Chat

## Run (web)

```bash
npm install
npx expo start --web
```

```bash

```

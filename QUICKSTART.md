## Garage App — Quickstart (Expo Go)

### Install

```bash
npm install
```

### Run on your phone (no Xcode / Android Studio)

```bash
npx expo start --tunnel
```

- iOS: scan QR with Camera → open in Expo Go
- Android: Expo Go → Scan QR code

### Notes
- **Maps** render on iOS/Android. Web shows a placeholder.
- If you get stuck, run:

```bash
npx expo-doctor
```
